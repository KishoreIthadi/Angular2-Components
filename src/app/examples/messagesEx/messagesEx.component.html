Component Code

You have to import and inject MessagesService into your component constructor and use it as shown below.The MessagesService have set method which takes two arguments message and CSS class. By default, the component comes with four classes messages_success, messages_danger, messages_warning & messages_info. You can pass your own classes and override the existing styles. 

    
        import { MessagesService } from 'angular2-comps'; 
        
        constructor(private messagesService: MessagesService){ } 
        
        click() {
        this.messagesService.SetMessage("Success Message", "messages_success");
        this.messagesService.SetMessage("Warning Message", "messages_warning");
        this.messagesService.SetMessage("Error Message", "messages_danger");
        this.messagesService.SetMessage("Info Message", "messages_info");
        }
