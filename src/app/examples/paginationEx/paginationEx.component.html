Markup

        totalrecords="103" 
        pagesize="10"
        pagescount="5" 
        selectedpage="1"
        (onPageChange)="onPageChange($event)" 
        previousbtncss="btn btn-primary glyphicon glyphicon-menu-left"
        nextbtncss="btn btn-primary glyphicon glyphicon-menu-right"
        btncss="btn"
        selectedcss="btn-primary"
        cssclass="custompagination">
        </pagination>

Component Code

The component comes with the onRatingSelected event as you can see in the above markup. This event will fire whenever the user selects a page/next/previous buttons. This event has one argument which will return selected page number. This event can be used to fetch records from DB based on the selected page number.

The style properties(previousbtncss, nextbtncss, btncss, selectedcss, cssclass) are optional. If user dosent provide with any value the default will be applied.
