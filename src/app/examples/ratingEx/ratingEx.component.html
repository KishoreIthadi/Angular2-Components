Rating


Markup

    
        <rating
        rating ="3.5" 
        starCount ="5"
        ratingID ="10"
        displayratingtext ="true"
        ratingcss= "rating"
        halfstarcss= "fa fa-star-half-o"
        fullstarcss= "fa fa-star"
        emptystarcss= "fa fa-star-o"
        (onRatingSelected) = "onRatingSelected($event)" 
        disable ="false">
        </rating>

Component Code

The component comes with the onRatingSelected event as you can see in the above markup. This event will fire whenever the user selects a rating. This event has one argument which will return selected rating and selected ratingID(if any). This event can be used write logic to update the DB. The rating property of the component can be updated in this event as rating value can also be average rating.

The style properties(ratingcss, halfstarcss, fullstarcss, emptystarcss) are optional. If user dosent provide with any value the default will be applied.
